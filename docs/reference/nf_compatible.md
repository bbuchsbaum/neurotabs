# Check strict concatenation compatibility

Check strict concatenation compatibility

## Usage

``` r
nf_compatible(a, b)
```

## Arguments

- a:

  An [nftab](nftab.md) object.

- b:

  An [nftab](nftab.md) object.

## Value

A list with `compatible` (logical) and `reasons` (character vector of
incompatibilities). Returned invisibly.
