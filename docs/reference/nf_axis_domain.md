# Declare an axis domain

Declare an axis domain

## Usage

``` r
nf_axis_domain(id = NULL, labels = NULL, size = NULL, description = NULL)
```

## Arguments

- id:

  Unique identifier for the label set (e.g., `"desikan68"`).

- labels:

  Relative path to a label table (TSV).

- size:

  Expected axis length.

- description:

  Human-readable description.

## Value

An `nf_axis_domain` object.
