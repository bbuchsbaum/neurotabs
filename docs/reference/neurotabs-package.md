# neurotabs: Neuro Feature Tables - Storage-Independent Neuroimaging Datasets

Reference implementation of the NFTab specification for row-oriented
neuroimaging datasets. Read, write, validate, and query collections of
neuroimaging observations with design metadata and feature values that
resolve to logical arrays (volumes, ROI vectors, surfaces) independent
of physical storage backend.
