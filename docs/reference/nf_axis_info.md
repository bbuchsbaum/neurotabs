# Axis metadata for a feature axis

Axis metadata for a feature axis

## Usage

``` r
nf_axis_info(x, feature, axis)
```

## Arguments

- x:

  An nftab object.

- feature:

  Feature name as a string or unquoted symbol.

- axis:

  Axis name from the feature's logical schema.

## Value

An `nf_axis_domain` object.
