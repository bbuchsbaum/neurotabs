# pkgdown/

## Purpose
Custom CSS and JavaScript overrides for the pkgdown documentation site. These files extend or override the default pkgdown Bootstrap theme.

## Key Files

| File | Description |
|------|-------------|
| `extra.css` | Custom stylesheet overrides applied to the pkgdown site |
| `extra.js` | Custom JavaScript applied to the pkgdown site |

## For AI Agents

### Working In This Directory
- Changes here only affect the rendered documentation site (`docs/`), not package functionality.
- `docs/` is auto-generated by pkgdown — do not edit files there directly.
- Site structure is configured in `_pkgdown.yml` at the project root.
- Rebuild the site with: `Rscript -e 'pkgdown::build_site()'`

### Testing Requirements
- No automated tests. Verify visually by building the site locally.

## Dependencies

### External
- `pkgdown` — site generator
- `albersdown` — custom theme (via `Config/Needs/website` in DESCRIPTION)
